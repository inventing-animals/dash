# Introduction

Dash is an information dashboard application built with [Avalonia](https://avaloniaui.net/) and [Ink](https://github.com/inventing-animals/ink). It can run as a local desktop application or be deployed on a server with multi-user support.

## Running locally (desktop)

```bash
dotnet run --project src/Dash.Client.Desktop/Dash.Client.Desktop.csproj
```

## Running the WASM demo locally

```bash
dotnet run --project src/Dash.Client.Browser/Dash.Client.Browser.csproj
```
